Release Notes for EROS 1.1

(4/18/2001, 5:00 pm EST) If you are looking for installation and build instructions, they can now be found here.

This is the first GPL'd release of the EROS system. To the best of my knowledge, all of the content on this CD may be redistributed under GPL or similarly non-restrictive (e.g. BSD) terms.

This release is not a runnable system -- it is more at the "Hello, world" stage. In particular, the eros/src/base/sysimage/ directory does not build a useful image at this time. My hope is that with contributions from the community this will soon improve.
