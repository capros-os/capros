EROS-Related Names

The name EROS started as a joke, with the expectation that it would change as things got further along. Like so many jokes, this one got out of hand. Some poor soul went and named some of the subsystems thematically, and it looks like we may stick with it. Some favorites appear below.

  Agape The wide-area, scalable transaction processing system. (Credit: Jon Shapiro)  
  Philos The document manager. (Credit: Jon Shapiro)  
  Voyeur The EROS debugger. (Credit: Norm Hardy)  
  SafetyGlass95 The Windows-95 emulator. Ours won't crash, but like the real thing it may take forever to release. (Credit: Neil Chase)  
  Dionysix A UNIX operating environment.  
  'SNOCRASH The distribution architecture. (Credit: Dave Farber)
