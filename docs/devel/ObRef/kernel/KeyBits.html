Kernel Objects

KeyBits

D R A F T

Description

The KeyBits key conveys the authority to extract the bits of another key. This authority compromises system security by exposing facts about the physical location of the objects on the disk, and is therefore closely held.

KeyBits returns the version of the keybits key and a 16 byte string which is the content of the key fields expressed as data.

Operations

Translate (OC = 1)

Returns the fields of the key as data.
Request SK0 The key to be examined.
Reply W The version of the KeyBits key. Later versions may return more information.
W Non-zero if the key is currently valid (future versions of keybits may expose internal information associated with rescinded keys).
[W : 4] The content of the key.
Result RC_OK Operation completed successfully.
