setboot Reference


 

Name

setboot -- maintain boot block table

Synopsis

setboot -l volume

setboot -g volume

setboot [-w] volume

Description

The setboot utility displays and alters the block table in the boot sector.

With -l, setboot displays the block table that is on the boot sector of the volume.

With -g, setboot displays the actual geometry of the volume.

The third form recomputes the block table from the actual geometry of the volume and displays the result. If -w is specified, it writes the result to the volume's boot sector.
